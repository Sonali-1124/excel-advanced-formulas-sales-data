# 📈 Advanced Excel Formulas Practice – Sales Dataset

## 📊 Project Overview
This project demonstrates the use of advanced Excel formulas on a fictional sales dataset. It includes business scenarios solved using functions like `VLOOKUP`, `INDEX-MATCH`, `IF`, `IFS`, `SUMIFS`, `COUNTIFS`, `TEXT`, `DATE`, and more.

## 🧠 Objectives
- Improve Excel formula writing skills
- Practice real-world sales analysis scenarios
- Build a reference file for HR or business reporting use

## 🧰 Tools Used
- Microsoft Excel (Office 365 or 2016+)
- Synthetic Sales Dataset (Excel)
